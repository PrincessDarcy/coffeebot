CoffeeBot is currently being recoded to work with Discord.Net CORE, so a lot of functionality is currently unavailable or being changed

CoffeeBot is my first real solo project, which began on March 3rd, 2016. The idea behind the project was to initially get myself situated and comfortable with the language and the API I was working with, but it expanded way beyond that!

CoffeeBot is a pseudo-AI coded in C# .NET using an unofficial Discord API called Discord.Net. Discord is a text and voice client, similar to Skype or Teamspeak. The product is still in development but is showing very promising results with what it is currently capable of. I also have an interface that allows me to use CoffeeBot commands outside of servers where she is located. At this point in time, CoffeeBot consists of over 1,900 lines of code. Before anyone asks, I will not under ANY circumstances release any source code for this project.

CoffeeBot is currently launched and maintained on 3 servers, with a user count ranging from 12 to over 150 users. She is equipped with numerous commands and featured listed on the right, with more features added weekly!

Features currently slated for development include youtube audio streaming capabilities, a comprehensive language filter with a toggle option for specific channels, and the ability to change certain server settings based on role levels.

Features!

  • Some interaction messages. Sometimes CoffeeBot will react to certain things you say

  • Extensive tag filtering system, and the ability to toggle channels SFW/NSFW

  • All chat channels are logged, including timestamps

  • Specialized greeting messages for new users joining the server

User Commands!

  • !blep - Displays blep Darcy

  • !boop - Coffee boops you....

  • !boop - CoffeeBot boops a user

  • !coffee - Coffee?

  • !comcast - Dontcha know

  • !commands - Prints Coffeebot's commands!

  • !darcycon - Get your insulin ready

  • !find - Searches Derpibooru for an image matching the provided tags

  • !hidensfw - Toggles NSFW channels off

  • !info - Displays when user joined the server

  • !menu - Display's the menu to order from

  • !order - Would you care for something to drink?

  • !roll - Rolls a die. Eg. Roll 2d20

  • !serverinfo - Retrieves information about the server

  • !ship - Ships a chosen user with a random user

  • !ship - Ships you with a random user

  • !shownsfw - Toggles NSFW channels on

  • !tally - Shows menu order totals

  • !team alpha - Team Alpha - Awesome Reds

  • !team gamma - Team Gamma - Awesome Yellows

  • !team omega - Team Omega - Awesome Oranges

  • !throw - CoffeeBot throws things at someone

  • !throw - CoffeeBot throws things at you

Admins Commands!

  • !blacklist - Shou's Server Only - Adds user to the adult channel blacklist(Uses user's NAME)

  • !addnsfwtag - Adds to the server's nsfw filter

  • !addorder - Adds an order item to the database

  • !addsfwtag - Adds to the server's sfw filter

  • !listfilters - Prints SFW and NSFW filters for the server!

  • !listroles - Prints role and role IDs!

  • !quit - Will shut down CoffeeBot

Server Owner Commands!

  • !currentintro - Shows the current set intro message for the server.

Developer Commands!

  • !addblep - Adds a blep to the database

  • !additem - Adds an item to the database

  • !addquote - Adds a quote to the database

  • !removeitem - Removes an item from the database(Uses id number)

  • !removequote - Removes a quote from the database(Uses id number)

  • !togglelogging - Enables/Disables server logging
